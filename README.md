# TRABAJO-FINAL-HTML-CSS-MasterD

Este es un trabajo realizado para el Curso Superior en Programación de Páginas Web de MasterD. 
Se trata de una página de ejemplo de un fotógrafo, diseñada como un sitio web sencillo para mostrar su trabajo y facilitar su contratación.  

Al ser un ejercicio académico, este proyecto en específico está hecho únicamente con HTML y CSS.
